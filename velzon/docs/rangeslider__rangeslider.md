##### Overview[Official Website](https://www.npmjs.com/package/@angular-slider/ngx-slider)

Self-contained, mobile friendly slider component for Angular 6+ based on angularjs-slider.

###### Add Package

```
npm install --save ngx-slider-v2
```

###### Remove Package

```
npm uninstall --save ngx-slider-v2 or you can remove package by removing specific package from package.json
```
