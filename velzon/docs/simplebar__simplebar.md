2. Simplebar

##### Overview [Official Website](https://www.npmjs.com/package/simplebar-angular)

SimpleBar does only one thing: replace the browser's default scrollbar with a custom CSS-styled one without losing performances.

###### Add Package

```
npm install simplebar-angular --save
```

###### Remove Package

```
npm uninstall simplebar-angular or you can remove package by removing specific package from package.json
```
