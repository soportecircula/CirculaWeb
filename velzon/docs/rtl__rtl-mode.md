#### RTL Mode Setting

You need to uncomment the below code in the
`assets/scss/config/default/app.scss` file.

```
// @import "../../rtl/components-rtl";
// @import "../../rtl/layouts-rtl";
// @import "../../rtl/pages-rtl";
// @import "../../rtl/plugins-rtl";
```
